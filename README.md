# FESAC Course Materials Platform

## 📚 Project Overview
This system is a web-based platform developed to allow lecturers to upload course materials and students to access and download them efficiently.

## 🚀 Features
- User authentication (Student & Lecturer)
- Lecturer dashboard (Upload, Manage materials)
- Course creation (by level & semester)
- Student dashboard (View & download materials)
- Download tracking system
- API-based course loading (AJAX)

## 🛠️ Technologies Used
- PHP
- MySQL
- JavaScript (Fetch API)
- HTML/CSS

## 📂 System Modules
### Lecturer
- Upload materials
- Manage uploaded files
- Create courses

### Student
- View materials by course
- Download files
- View download history

## 🔧 Setup Instructions
1. Clone the repository
2. Import the database into phpMyAdmin
3. Configure database in `config/db.php`
4. Run project on localhost
